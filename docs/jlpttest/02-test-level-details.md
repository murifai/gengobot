## 5. Special Features

### 5.1 Grouped Questions (Unit-Based Randomization)

Questions that share common content (passages, audio, graphics) and must be kept together:

| Level  | Mondai with Grouping             | Percentage |
| ------ | -------------------------------- | ---------- |
| **N5** | 問題4, 5, 6 (3 mondai)           | ~19%       |
| **N4** | 問題4, 5, 6 (3 mondai)           | ~22%       |
| **N3** | 問題3, 5, 6, 7 (4 mondai)        | ~28%       |
| **N2** | 問題11, 12, 14 (3 mondai)        | ~10%       |
| **N1** | 問題9, 10, 11, 12, 13 (5 mondai) | ~38%       |

### 5.2 Question Unit Types

- **Cloze Test**: Fill-in-the-blank passages with numbered positions
- **Reading Comprehension**: Short passages with multiple questions
- **Long Reading**: Extended passages with multiple questions
- **A-B Comparison**: Two related texts with comparison questions
- **Info Graphic**: Posters, schedules, announcements with questions
- **Long Audio Multi**: Single audio file with 2+ questions

### 5.3 Special Scoring Features

#### N3 Dual Normalization

- **Reading Part** (問題1-3): 23 questions → normalized to 60 points
- **Grammar Part** (問題4-7): 16 questions → normalized to 60 points
- Both parts evaluated independently for pass/fail

#### N2/N1 Combined Sections

- Vocabulary and Grammar/Reading administered together
- Single timer for entire section (105-110 minutes)
- Separately normalized for scoring

## 6. Pass Requirements

### 6.1 Overall Pass Criteria

| Level  | Total Score Required | Sectional Minimum                           |
| ------ | -------------------- | ------------------------------------------- |
| **N1** | 100/180              | Each section ≥ 19/60                        |
| **N2** | 90/180               | Each section ≥ 19/60                        |
| **N3** | 95/180               | Each section ≥ 19/60                        |
| **N4** | 90/180               | (Vocab+Grammar) ≥ 38/120, Listening ≥ 19/60 |
| **N5** | 80/180               | (Vocab+Grammar) ≥ 38/120, Listening ≥ 19/60 |

### 6.2 Sectional Requirements

**N5 & N4**:

- Vocabulary + Grammar/Reading combined: Must achieve **38/120 points**
- Listening: Must achieve **19/60 points**
- Total: Must achieve minimum overall score

**N3**:

- Vocabulary: Must achieve **19/60 points**
- Grammar/Reading: Must achieve **19/60 points** (dual normalization)
- Listening: Must achieve **19/60 points**
- Total: Must achieve **95/180 points**

**N2**:

- Vocabulary/Grammar/Reading: Must achieve **19/60 points** (separately normalized)
- Listening: Must achieve **19/60 points**
- Total: Must achieve **90/180 points**

**N1**:

- Vocabulary/Grammar/Reading: Must achieve **19/60 points** (separately normalized)
- Listening: Must achieve **19/60 points**
- Total: Must achieve **100/180 points**

### 6.3 Reference Grades

After normalization, reference grades are assigned:

- **Grade A**: High proficiency (typically 67-100% of section max)
- **Grade B**: Moderate proficiency (typically 34-66%)
- **Grade C**: Basic proficiency (typically 0-33%)

**Note**: Reference grades are informational only and do not affect pass/fail status.
