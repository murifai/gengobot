# JLPT Mondai Default Explanations

This document contains the **standardized** explanation text for each mondai (問題) across all JLPT levels.

## Important Notes

⚠️ **All questions within the same mondai share the SAME explanation text**

This is intentional because:
- Questions are randomized during test generation
- The explanation describes the **type** of question (e.g., "choose the correct reading")
- It applies equally to all questions in that mondai, regardless of order

📌 **The explanation displays ONLY before the first question of each mondai**
- Students see the explanation when they start a new mondai
- As they navigate through questions in the same mondai, the explanation is hidden
- When they move to the next mondai, the new explanation appears

## Usage

When creating JLPT questions in the admin panel:
1. Identify the **level**, **section**, and **mondai number**
2. Copy the corresponding explanation from this document
3. Paste it into the `mondaiExplanation` field for **ALL questions** in that mondai
4. The explanation will automatically display at the top of each question

**Example**: All questions in N5 文字・語彙 もんだい１ should have the same explanation:
```
____の ことばは ひらがなで どう かきますか。１・２・３・４からいちばんいいものをひとつえらんでください。
```

### 文法・読解 (Grammar & Reading)

**文法 - もんだい１**
```
（ ）に何を入れますか。１・２・３・４からいちばんひといいものを一つえらんでください。
```

**文法 - もんだい２**
```
★ に入るものはどれですか。１・２・３・４からひといちばんいいものを一つえらんでください。
```

**文法 - もんだい３**
```
22 から 26 に 何を 入れますか。ぶんしょうのいみをかんがえて、１・２・３・４からいちばんいいものをひと一つえらんでください。
```

**読解 - もんだい４**
```
つぎの (1)から(3)のぶんしょうを読んで、しつもんにこたえてください。こたえは、１・２・３・４からいちばんいいものを一つえらんでください。
```

**読解 - もんだい５**
```
つぎのぶんしょうを読んで、しつもんにこたえてください。こたえは、１・２・３・４からいちばんいいものを一つえらんでください。
```

**読解 - もんだい６**
```
右のページを見て、下のしつもんにこたえてください。こたえは、１・２・３・４からいちばんいいものを一つえらんでください。
```

### 聴解 (Listening)

**もんだい１**
```
では、はじめにしつもんをきいてください。それからはなしをきいて、もんだいようしの１から４のなかから、いちばんいいものをひとつえらんでください。
```

**もんだい２**
```
では、はじめにしつもんをきいてください。それからはなしをきいて、もんだいようしの１から４のなかから、いちばんいいものをひとつえらんでください。

**もんだい３**
```

```

**もんだい４**
```
は、えなどがありません。ぶんをきいて、１から３のなかから、いちばんいいものをひとつえらんでください。
